# WhatsApp UI/UX Prototype

## Project Overview

Designed a WhatsApp-inspired mobile application prototype using Figma. The project focuses on creating a modern and user-friendly messaging experience with multiple interactive screens.

## Features

* User Login Screen
* OTP Verification Screen
* Home/Chat Screen
* Updates & Status Screen
* Calls Screen
* Profile Screen
* Settings Screen
* Interactive Navigation Flow

## Tools Used

* Figma

## Responsibilities

* Designed complete user interface layouts
* Created reusable UI components
* Developed user flow and navigation
* Maintained design consistency across screens
* Applied modern UI/UX principles

## Outcome

Successfully created a high-fidelity WhatsApp-inspired prototype demonstrating mobile application design skills and user experience planning.
